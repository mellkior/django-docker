# django-docker
A dockerized django 4 project.

## Running locally

### Create a virtual environment.
```
python -m venv .venv
```

### Activate the virtual environment.
> Bash
```
. .venv/bin/activate
```
> Powershell
```
.venv\Scripts\Activate.ps1
```
> CMD (Windows)
```
venv\Scripts\activate.bat
```
### Install django
```
pip install -r requirements.txt
```
